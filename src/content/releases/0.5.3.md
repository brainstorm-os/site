The one where your old notes arrive intact. If you imported an Anytype space
before, some pasted screenshots showed up broken — and a few images quietly
showed the **wrong** picture. The importer now reads the export's own file
map, so every attachment binds exactly: **457 of 457** in our own three-year
test vault. Re-running an import repairs what's already there — no fresh
vault needed.

Around the import, a round of day-two fixes: Files' Gallery and Grid tiles
fill their cells with real thumbnails, imported notes recover their titles on
first open, option menus stop truncating their labels into guessing games,
the Agent renders nested lists properly, and connecting an IMAP mailbox no
longer fails because your password manager pasted a trailing space.
