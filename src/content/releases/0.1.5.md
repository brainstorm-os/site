Brainstorm v0.1.5 is the first public beta — the desktop shell, your local-first
vault, the bundled first-party apps, and end-to-end encrypted multi-device sync,
in signed and notarized builds for macOS, Windows, and Linux.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Updates install in-app from **Settings → Updates**.

It's a beta: expect rough edges, and keep backups of anything important. Your
vault is a plain folder you fully control.
