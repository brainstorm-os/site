The first minor since the analytics train, and the biggest release since the
public beta opened. If you're coming from Anytype, your whole space moves in:
Settings → Backup & Migration accepts the export (folder or zip) and your
pages arrive as real, editable documents — structure, dates, tags and links
intact. And from this version on, Brainstorm checks for updates itself and
offers them as a toast: this is the last release you'll have to go looking for.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
