The follow-up 0.5.0 promised: if you brought your Anytype space over, your
screenshots and PDFs are now actually *in* your documents — verified against a
real 460MB space, twice, without duplicating a thing. Database grows rollup
columns, the Agent can put an answer into a note (only when you ask it to), and
every drag in the app has a keyboard path now.

If you're on 0.5.0, the update toast will offer you this one — that's new too.

macOS builds are signed with a Developer ID and notarized by Apple. Windows is
currently unsigned (SmartScreen: **More info → Run anyway**). Existing installs
update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
