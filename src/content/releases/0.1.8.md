Brainstorm v0.1.8 turns on semantic search. A local embedding model runs
entirely on your device — searching "vacation ideas" surfaces the note titled
"places to visit this summer", with nothing sent off your machine. The model
downloads once on first run (search works normally in the meantime, with a
progress indicator in the new activity center), and indexing is incremental:
edits re-index only what changed, so large vaults stay fast.

Around the shell, the dashboard gains a background-activity center that
collects long-running work — model downloads, indexing, sync — in one place,
plus six new widgets with size presets and keyboard move & resize. Chat gets a
richer composer with formatting and inline @-mentions in both chat and agent
messages. The editor learns templates: insert one from the slash menu, or save
any document as a template. Bookmarked articles now keep their images, stored
encrypted with the rest of your vault, and onboarding gets a fresh look with
the new Rose theme. Accent-colored and muted text now meet WCAG AA contrast in
every theme.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
