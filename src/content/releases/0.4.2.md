Presence lands on the graph canvas. When you share a graph, you now see collaborators' cursors and selection rings move live — the same real-time awareness Whiteboard already had. Database joins the fleet too: the header shows who's viewing the same record.

App localization takes a big step forward. Switch to Spanish in Settings → Language and chrome in twelve apps updates live without a restart — Code editor, Browser, Preview, Chat, Agent, Contacts, Mailbox, Books, Theme editor, Form designer, Automations, and Files.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
