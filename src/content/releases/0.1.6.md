Brainstorm v0.1.6 is a polish release on the public beta. PDFs now open fit to
the page (and re-fit as the window resizes) instead of rendering at 100% and
clipping; text fields, selects, and buttons sit on one shared control height so
toolbars and forms line up across apps; Books shows a single empty state with
one clear action; and the entity inspector reads as a denser, consistent
property sheet.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
