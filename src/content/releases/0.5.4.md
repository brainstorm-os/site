The one where Mailbox becomes a mail client you'd actually live in. The
compose window grew up into a real editor — bold, lists, quotes, links —
and what you send is proper HTML mail with a plain-text fallback. Your
accounts are finally visible things: each one sits in the sidebar with
Sync and Remove at hand, and when a sync fails you get a readable
explanation and a one-click **Reconnect** instead of a dead end.

Reading mail got the same pass. One click opens a message (single mails no
longer masquerade as one-item conversations), folders show their names
instead of `INBOX/` server paths, blocked remote images stop painting
broken glyphs — and the "Show remote content" button, which a hidden
second security policy had silently neutered, now actually shows it.
Connecting an IMAP account also asks how much history to sync, from 30
days to everything.

Beyond mail: the built-in Browser now presents a standard Chrome identity,
so strict sites like X.com stop rejecting sign-ins they think are bots, and
a design-system pass put every corner radius on tokens with one enforced
size for hero buttons.
