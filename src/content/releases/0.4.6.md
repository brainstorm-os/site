A quick patch on top of 0.4.5, mostly bug fixes. The shared editor theme had
drifted from Notes, so every other app hosting an editor showed broken callouts,
a vertical inline toolbar, and unstyled titles — that's synced back up. The
Files grid stops overlapping tiles after a repack, widget empty states sit
centred again, and the dashboard now shows its snap grid while you drag an icon.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
