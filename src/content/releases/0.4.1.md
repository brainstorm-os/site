Live presence spreads across the fleet. Avatar stacks now show in every shared-entity app header — Tasks, Chat, Journal, Calendar (when editing an event), Graph, Code editor, Preview, and Books — not just Notes and Whiteboard. When you share something, you can see who else is there at a glance.

The in-app browser no longer leaves a blank view or bleeds content between tabs when you switch. Settings gets a few polish passes: the Identity section shows a copyable fingerprint chip, secondary text is easier to read, and the Updates card layout is tidied.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
