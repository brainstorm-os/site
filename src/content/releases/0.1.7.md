Brainstorm v0.1.7 is a fixes release on the public beta, mostly from a full pass
over every app. Status and select-style cells in the Database now edit as a
type-or-pick combobox — filter the values already in that column or commit a new
one — instead of dropping you into a blank text field. The calendar draws
multi-day events as a single continuous bar rather than repeating a clipped
title in each day; Preview centers its empty state and docks the details panel
on the right; and chat messages no longer surface as untitled rows that bury
your documents in the Files and Database browsers.

The assistant now renders the documents it cites as clickable titles instead of
raw internal ids, and the graph labels untitled items by type. Smaller polish
lands too: consistent controls in the Agent's settings, clearer empty states in
Automations and Contacts, and task date chips that read correctly to screen
readers.

macOS builds are signed with a Developer ID and notarized by Apple, so they open
without a Gatekeeper warning. Windows is currently unsigned (you may see a
SmartScreen "unknown publisher" prompt — choose **More info → Run anyway**).
Existing installs update in-app from **Settings → Updates**.

It's still a beta: keep backups of anything important. Your vault is a plain
folder you fully control.
